# ポートフォリオサイト
HTML、CSS、JavaScriptを使用して、ポートフォリオサイトを作成しました。

- トップページ、自己紹介ページ、作品ページを作成しました。
- カーソルを円形に変更し、リンクホバー時はサイズが大きくなりクリックできることをわかりやすくしています。
- ロゴ以外の文字装飾は画像に頼らず、CSSを使用しています。
